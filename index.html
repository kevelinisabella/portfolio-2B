Minha Biblioteca: Uma Webpage Personalizada

As redes sociais são plataformas online que conectam pessoas e organizações através de perfis, permitindo o compartilhamento de conteúdo e interação. Elas funcionam como um espaço virtual para comunicação e troca de informações.

O DogZap seria uma rede social específica para amantes de cães. Nele, os usuários se conectariam, compartilhariam fotos e experiências de seus pets, e encontrariam recursos relevantes, tornando-o um espaço dedicado ao universo canino.

a escolha do roxo e amarelo/dourado para o DogZap parece ter sido estratégica para criar um aplicativo visualmente atraente, fácil de usar e que transmita uma sensação de alegria, qualidade e cuidado, elementos que se alinham bem com um público apaixonado por cães.

projeto DogZap no Figma demonstra um bom planejamento de fluxo do usuário (boas-vindas > login > funcionalidade principal de chat), com uma identidade visual forte e um foco claro no público-alvo de amantes de cães, utilizando elementos gráficos e textuais que reforçam o tema canino.

Minha Biblioteca: Uma Webpage Personalizada

Este projeto de robótica usa um Arduino Uno e um protoboard para controlar múltiplos LEDs coloridos. Fios jumpers conectam os pinos digitais do Arduino aos LEDs (com resistores para proteção) e ao terra comum. Através de programação no Arduino, os LEDs podem ser acionados em sequências ou padrões, servindo como uma introdução fundamental à eletrônica e programação.

Os materiais essenciais para este projeto foram: um **Arduino Uno** para controle, um **protoboard** para montagem sem solda, **LEDs** de várias cores para a saída visual e **resistores** para protegê-los. **Fios jumpers** foram usados para as conexões entre os componentes e o Arduino.

Este projeto é uma porta de entrada para eletrônica e programação com Arduino. Ele ensina a controlar saídas digitais, ligando e desligando LEDs através de código. Essencialmente, ajuda a entender como montar circuitos e desenvolver a lógica de programação para criar sequências de luzes. É a base para projetos futuros mais elaborados.

Este projeto de robótica usa um Arduino Uno e um protoboard para controlar um LED RGB. Através de três pinos PWM e resistores, é possível variar a intensidade de cada cor (vermelho, verde, azul) do LED. Isso permite criar uma vasta gama de cores e efeitos luminosos, sendo uma excelente prática para entender PWM e mistura de cores.

Este projeto didático utiliza um Arduino e um LED RGB para ensinar a controlar a intensidade e a mistura de cores de luz. Serve para praticar a lógica de programação, o uso de PWM e a montagem de circuitos. É uma base essencial para criar efeitos visuais dinâmicos e desenvolver projetos de iluminação e sinalização mais complexos.

Os materiais utilizados neste projeto foram: um **Arduino Uno** para controle, um **protoboard** para montagem rápida, um **LED RGB** para efeitos de cor, **resistores** para proteção do LED e **fios jumpers** para as conexões.
